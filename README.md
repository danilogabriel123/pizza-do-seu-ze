# 🍕 Pizzaria do Zé — Landing Page

**Nome sugerido para o repositório:** `pizzaria-do-ze`

## Sobre o projeto

A Pizzaria do Zé é uma landing page responsiva criada para uma pequena pizzaria de bairro que não tinha presença na internet — sem um site, o estabelecimento perdia clientes porque as pessoas não tinham como ver o cardápio com preços, horários de funcionamento ou informações de contato de forma rápida, então a página resolve esse problema apresentando o negócio de maneira organizada, com um cardápio interativo onde o cliente adiciona ou remove itens, acompanha o resumo do pedido em tempo real e finaliza o contato diretamente pelo site.

## Como abrir

Baixe os 3 arquivos na mesma pasta e abra o `index.html` no navegador. Não precisa instalar nada.

## Arquivos

- `index.html` — estrutura da página com todas as seções
- `style.css` — estilos visuais, cores e responsividade
- `script.js` — lógica dos botões de quantidade e cálculo do total

## Funcionalidades

- Header fixo com navegação
- Banner de apresentação
- Seção "Sobre nós" com dados do negócio
- Cardápio com 6 produtos e controles de quantidade
- Resumo do pedido atualizado em tempo real
- Layout responsivo (celular e desktop)
